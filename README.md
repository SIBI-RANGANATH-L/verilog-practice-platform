## How to Run Locally

###  Backend (Spring Boot)
```bash
cd verilog_backend
./mvnw spring-boot:run
````

###  Frontend (React)

```bash
cd verilog-ui
npm install
npm start
```

Then open:

 `http://localhost:3000` ‚Äì React UI 

 `http://localhost:8080` ‚Äì Spring Boot API
